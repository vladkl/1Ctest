Список запросов

Список запросов отображает все выполнения запросов, относящиеся к данной вершине дерева анализа проблем. Запросы в списке отсортированы по убыванию веса. Список выполнений запросов имеет следующие поля:

  • вес данного выполнения запроса;
  • время выполнения запроса;
  • контекст запроса, то есть строка кода конфигурации, из которой был выполнен запрос;
  • SQL-текст запроса (только начальный фрагмент).

Двойной щелчок по строке списка открывает форму с подробной информацией о данном выполнении запроса.
