Выбор сценария работы

«Центр управления производительностью» имеет три предопределенных сценария:

  • Мониторинг;
  • Регламентный мониторинг;
  • Просмотр.

Мониторинг

Сценарий мониторинга подразумевает «онлайновое» подключение к исследуемой базе и предназначен для:

  • мониторинга текущих значений выбранных показателей производительности;
  • включения \ выключения записи (сохранения в базе ЦУП) текущих значений показателей производительности для их последующего («оффлайнового») анализа;
  • включения \ выключения сбора дополнительной информации (аналитических показателей) для дальнейшего анализа имеющихся в системе проблем производительности.

Для того чтобы ЦУП смог подключится в режиме мониторинга к исследуемой информационной базе, версии платформы 1С:Предприятия, на которых запущен ЦУП и исследуемая база, должны совпадать.

Регламентный мониторинг

Сценарий регламентного мониторинга похож на режим "Мониторинга", только вся работа сценария автоматизирована и не требует присутствия эксперта. Сценарий регламентного мониторинга производительности автоматически анализирует текущее состояние системы и в зависимости от этого выполняет необходимые действия.

Если значения показателей производительности незначительно ухудшадся, мониторинг станет более интенсивным и "онлайновые" показатели начнут записываться в базу ЦУП.

Если значения показателей производительности ухудшадся значительно, то начнется автоматический сбор аналитических данных для последующего анализа.

Если значения показателей начнут улучшаться, интенсивность анализа вернется в начальное состояние.

Эксперт может в любое время открыть исследуемую базу в режиме просмотра и посмотреть, что происходило в системе.

Просмотр

Сценарий просмотра не требует подключения к исследуемой базе и используется для решения следующих задач:

  • просмотр сохраненных значений показателей производительности за любой период;
  • анализ технической информации об имеющихся проблемах производительности.

Для выбора режима работы нажмите кнопку «Новое соединение Ctrl+N» в панели инструментов. Из списка сценариев выберите нужный и нажмите «ОК».
