Работа в режиме мониторинга

Общие сведения

Основная форма мониторинга значений показателей производительности состоит из двух взаимосвязанных областей:

  • текущие значения показателей производительности;
  • список выбранных показателей производительности.

Область текущих значений показателей производительности отображает в графическом виде значения всех выбранных показателей производительности. Если режим записи показателей выключен, то показатели, выходящие за границы отображаемой области «забываются».

В режиме мониторинга ЦУП не требует больших ресурсов от компьютера, на котором запущен, и не создает заметной нагрузки на исследуемую информационную базу даже при выборе всех показателей производительности (за исключением аналитических показателей). Поэтому выбор интервала опроса диктуется почти исключительно удобством пользователя ЦУП.

Область текущих значений показателей производительности отображает точное время начала и окончания интервала.

Область списка выбранных показателей производительности отображает список всех показателей производительности. Для каждого показателя производительности отображаются следующие данные:

  • параметры отображения показателя:
    • цвет;
    • cтиль линии;
    • толщина линии.
  • включена ли запись значений показателя (флажок);
  • единицы измерения;
  • масштаб отображения;
  • последнее полученное значение показателя;
  • агрегатное значение показателя (из тех значений, которые входят в отображаемую область):
    • минимум;
    • среднее;
    • максимум;
    • сумма.

Пользователь может настраивать состав показателей и параметры их отображения:

  • добавить к списку выбранных показателей новый показатель;
  • удалить показатель из списка выбранных показателей;
  • настроить параметры отображения показателя.

Настройка списка показателей производительности осуществляется при помощи панели инструментов в нижней части формы.

Кроме этого, вы можете включить режим «подсветки» графика выбранного показателя. Для этого следует активировать кнопку с пиктограммой «лампочка». Когда данный режим включен, при выборе показателя в списке его график будет отображаться черным цветом.

Работа с показателями производительности

ЦУП позволяет сохранять текущие значения показателей производительности в своей информационной базе для их дальнейшего анализа.

Вы можете включать и выключать сохранение любых показателей в любое время и на любой период.

ЦУП позволяет собирать и анализировать подробную техническую информацию о следующих проблемах производительности:

  • медленная работа запросов СУБД;
  • ожидания на блокировках – простои системы из-за ожиданий на блокировках данных;
  • взаимоблокировки данных – неразрешимые конфликты блокировок.

Список показателей ЦУП содержит группу «Анализ», в которую входит три показателя производительности:

  • «Анализ запросов»;
  • «Анализ ожиданий на блокировках»;
  • «Анализ взаимоблокировок».

Работа в режиме просмотра

Общие сведения

Режим просмотра предназначен для:

  • просмотра сохраненных значений показателей производительности за любой период;
  • анализа технической информации об имеющихся проблемах производительности;

ЦУП в режиме просмотра отображает ту информацию, которая была записана в режиме мониторинга. Работа в режиме просмотра не требует подключения к информационной базе.

Для начала работы в этом режиме необходимо выбрать сценарий «Просмотр» и указать информационную базу аналогично режиму мониторинга. Будет открыто основное окно «Центра управления производительностью», которое будет отображать сохраненные значения показателей производительности:

Пользователь так же может добавлять и удалять показатели производительности.

Удаленные показатели производительности будут удалены с графика. При этом их значения из информационной базы ЦУП не пропадут. В дальнейшем пользователь может добавить эти показатели в список. Добавить в список можно только те показатели, которые имеют сохраненные значения на выбранном интервале времени.

Выбор интервала просмотра

Пользователь может просматривать всю сохраненную информацию (полный интервал просмотра – от начала до конца данных - выбирается по умолчанию) или выбрать какуюто его часть – ту, которая интересует пользователя в настоящий момент.

Выбор масштаба отображения

Пользователь может просматривать выбранный интервал целиком, либо произвольным образом настраивать масштаб отображения.

Это может быть необходимо для того, чтобы более детально рассмотреть какой-либо участок графика.

Пользователь может выбирать масштаб вручную, увеличивая или уменьшая его в два раза. Пользователь так же может указать одно из предопределенных значений масштаба, выбрав его из списка («минута», «час» и т.д.).

Если масштаб отображения меньше интервала просмотра, пользователь может перемещаться в пределах заданного интервала при помощи полосы прокрутки, расположенной под графиком.

Другие возможности режима просмотра

Удаление данных

Пользователь может удалить все данные, которые находятся в видимом интервале.

Обратите внимание на то, что при выборе этого действия будут удалены все показатели производительности, находящиеся в видимом интервале времени (в том числе те показатели, которые не отображены на графике).

Анализ проблем производительности

Если во время мониторинга записывались аналитические показатели, то пользователь может перейти к анализу проблем производительности. Для этого необходимо нажать кнопку «Анализ».

Форма анализа проблем будет отображать данные в соответствии с текущими настройками. А именно:

  • будут отображены только те аналитические показатели, которые попали в просматриваемый интервал времени;
  • будут отображены только выбранные аналитические показатели.

Таким образом, пользователь может выбирать, какие именно показатели и за какой период времени он хочет анализировать в данный момент.
