Настройка параметров мониторинга

Частота опроса значений показателей зависит от настроек режима мониторинга.

Значение по умолчанию для интервала опроса = 1 секунда. При таком значении настройки ЦУП будет получать значения выбранных показателей производительности один раз в секунду, а график будет отображать значения 60 секунд (1 минута). Это минимальное значение интервала опроса.
