Информация о взаимоблокировке

Форма содержит данные о последовательности установки блокировок каждым процессом. Кроме этого, она содержит некоторую дополнительную техническую информацию:

  • полный XML-текст исходной информации о взаимоблокировке, получаемой от MS SQL Server – так называемый граф взаимоблокировки (deadlock graph);
  • дополнительные параметры взаимоблокировки.
